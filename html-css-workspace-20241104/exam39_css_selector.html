선택자란?
특정한 HTML 태그 요소를 선택할 때 사용하는 표기 방식
원하는 태그를 선택하여 원하는 스타일을 각각 적용할 때


HTML 안의 모든 요소를 선택할 때 {}만으로 선택할 수 있다.

태그 작성자

HTML 문서 내의 동일한 태그 모두에게 스타일을 적용시키고자 할 때 사용한다. 

p{color:blue}와 같은 형태로 사용한다.

아이디 선택자

HTML문서 내의 선택한 특정 아이디에 해당하는 요소의 스타일을 설정할 때 사용한다. 

#아이디명{color:red}

클래스 선택자

HTML문서 내에 특정 그룹을 나타내는 요소들을 하나의 스타일로 묶어 선택할 때 사용한다. 

.클래스명{color:green;}
